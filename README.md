# Craftable Elytra

Creates an elytra crafting recipe.

The recipe requires:

-   3 Shulker Shells
-   2 Phantom Membranes
-   2 Purpur Blocks
-   1 Diamond
-   1 Eye of Ender

# XPBegone

Allows `keepInventory` to be active while losing experience points on death.

Automatically sets all dimension to have `keepInventory true`
